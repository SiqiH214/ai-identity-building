## 🚀 如何使用

### 快速启动（3 步）

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量（创建 .env.local）
GEMINI_API_KEY=your_api_key_here

# 3. 启动开发服务器
npm run dev
```

### 访问应用
打开浏览器访问: http://localhost:3000
