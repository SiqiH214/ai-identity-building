## ❓ 常见问题

### Q: 手机无法访问？
A: 
1. 确认手机和电脑在同一 WiFi
2. 检查防火墙设置
3. 尝试重启应用和浏览器

### Q: IP 地址改变了？
A: 
- WiFi 重连后 IP 可能变化
- 运行 `ifconfig | grep "inet "` 查看新 IP
- 使用新 IP 访问

### Q: 如何让外网访问？
A:
- 需要配置路由器端口转发
- 或使用 ngrok 等内网穿透工具
- 不推荐直接暴露到公网

### Q: 性能如何？
A:
- 局域网速度很快（通常 100Mbps+）
- 图片加载几乎无延迟
- API 调用速度与本机相同
